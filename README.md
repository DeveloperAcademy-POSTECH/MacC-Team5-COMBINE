# 🗂️ 프로젝트 소개
초등학교 1-3학년 저시력 시각 장애 어린이들을 위한 전래 동화와 연계한 코딩 교육 서비스입니다.

<br/>

# 🛠️ 기술 스택
- 화면 구성: UIkit(Code-Based)
- 디자인 패턴: MVVM
- 비동기 프로그래밍: Combine, GCD
- 데이터베이스: SwiftData
- 애플 고유 기술: UIAccessibility, CoreMotion, CoreHaptic, 
- 사용툴: 노션, 깃, 피그마
